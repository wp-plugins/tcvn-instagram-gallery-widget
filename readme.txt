== Description ==

The TCVN Instagram Gallery Widget for WordPress helps you display your images from Instagram in a clean and simple gallery. Showcase your latest Instagram images on your WordPress website, or share the Instagram images you like. Easy to set up and configure, and fully customizable using templates.

Features:

- Support multi Wordpress Version.
- Load your latest images shared on Instagram.
- Load the last images you liked on Instagram.
- Load the latest images in your feed from the users you follow.
- Load The most popular images on Instagram.
- Your followers on Instagram.
- Latest images with a tag name.
- Optional Display Loading Images.
- Languages included: English
- Allow loading wordpress widget inside post, page.
- To manipulate almost everything !
- Option to disable jQuery loading again
- Totally customizable and skinnable using HTML and CSS
- Lightweight, modern and fast-loading design.
- W3C XHTML 1.0 Transitional. W3C CSS Valid.
- Tableless design and 100% CSS-based
- Fully compatible IE7+, Firefox 2+, Flock 0.7+, Netscape, Safari, Opera 9.5
- Easy installation

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the entire 'tcvn-instagram-widget' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'TCVN Instagram Gallery Widget' to your sidebar
